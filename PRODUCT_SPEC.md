# Features

## Main list view

The main list view should have the following:
- the habit name
- then the date and time (in the browser's local time)
- then a string of `X`s corresponding to how many times that habit 
  has been done
- the button to increment the number of times performed


## The Habit View

- should have most of the info, but leave off the following:
  - habit ID
  - user ID
- it doesn't need the raw timestamp -- it only needs the local time of the last time performed
- the habit deletion button shouldn't be a `-` sign, it should be a button that says `delete`
- the items don't need to say "last performed" when giving the date and time --
  the date and time (in browser/local) can just be there -- people know what it means
- we don't need a title at the top that says "Habit Tracker"
