# Project Notes

This is a lightweight place to keep context for the habit tracker prototype.

## Current Shape

- Single-page Vite app.
- Supabase stores the habit rows.
- No login yet.
- A fixed prototype user ID is used when inserting new habits.
- The data is just sample data, so it's okay to mutate

## Product Ideas

- Add a simple auth flow once the core habit tracking feels good.
- Add notes or categories to each habit.
- Add a streak or calendar view.
- Make the increment update atomic in Supabase before multiple users use it.

## Collaboration Notes

Use this file for rough notes, open questions, feature ideas, and things that feel awkward while you are using the app. Short messy notes are perfect.
